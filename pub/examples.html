Example Use of IBCG

IBCG creates realtime interactive multi state graphs where you can export and import states for anyboady learning to play around with the data.

Features of this release

  1. Auto normalization of datapoints to fit at 75% of the canvas height.
  2. Toggling between multiple graph workspaces.
  3. Importing and exporting a multi workspace state.
  4. Responsive graph visualization that adjusts datapoint width with respect to available space.

Example demonstrating the empty state when a Professor/TA wants to demonstrate graphs from scratch


new IBCG().init('#ibcg').render();
